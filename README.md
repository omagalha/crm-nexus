# Nexus CRM

MVP de CRM institucional para a Nexus Educacao, criado com React + Vite.

## Rodar localmente

```bash
npm install
npm run dev
```

## Estrutura

- `src/components`: componentes reutilizaveis da interface.
- `src/pages`: telas principais do CRM.
- `src/layouts`: layout autenticado com sidebar e header.
- `src/services`: camada inicial para API e servicos por dominio.
- `src/data`: dados mockados para o MVP.
- `src/styles`: tema, estilos globais e identidade Nexus.
